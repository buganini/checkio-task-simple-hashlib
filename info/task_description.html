In a correctly designed system, a plain password is not stored in the database. Instead, a hash of the password is stored. You are going to verify a password and you will need to use the same one-way function to calculate the hash and do a comparison.

Input: Two string arguments. One is the string to be hashed, another is the hash algorithm.

Output: String, hexadecimal hash for for input string using given algorithm

Example: 

	assert checkio(b'welcome', 'md5') == '40be4e59b9a2a2b5dffb918c0e86bhe3d7'
	assert checkio(b'happy spam', 'sha224') == '6e9dc3e01d57f1598c2b40ce59fc3527e698c77b15d0840ae96a8b5e'
